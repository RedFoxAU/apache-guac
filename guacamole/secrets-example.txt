# Secrets File Examples for Apache Guacamole
# ==========================================
# 
# This file shows examples of secret files that need to be created.
# 
# IMPORTANT: 
# - Create actual secret files in your ${WORKING_DIR}/guacamole/secrets/ directory
# - Use strong, unique passwords
# - Set proper file permissions (chmod 600)
# - Never commit actual secret files to git

# File: ${WORKING_DIR}/guacamole/secrets/postgres_password.txt
# Content: Your strong PostgreSQL password (no newline at end)
# Example: MyStr0ngP@ssw0rd!2024

# File: ${WORKING_DIR}/guacamole/secrets/ldap_password.txt (if using LDAP)
# Content: LDAP bind password
# Example: LDAP_Bind_P@ssw0rd

# File permissions should be set to 600 for security:
# chmod 600 ${WORKING_DIR}/guacamole/secrets/*.txt
